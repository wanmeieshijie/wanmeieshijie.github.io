# 用 Markdown 写博客

**2026年5月20日**

---

Markdown 是一种轻量级标记语言，让你专注于写作而不是排版。

## 基本语法

### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

### 强调

```markdown
*斜体* 或 **粗体** 或 ~~删除线~~
```

### 列表

```markdown
- 无序列表项 1
- 无序列表项 2

1. 有序列表项 1
2. 有序列表项 2
```

### 链接和图片

```markdown
[链接文字](https://example.com)
![图片描述](image.jpg)
```

### 代码

行内代码：`console.log('hello')`

代码块：
```javascript
function greet(name) {
  return `你好，${name}！`;
}
```

### 引用

> 这是引用块的内容。
> 可以有多行。

### 表格

| 语法 | 效果 |
|------|------|
| `**粗体**` | **粗体** |
| `*斜体*` | *斜体* |
| `` `代码` `` | `代码` |

## 为什么用 Markdown？

1. **简单** - 几分钟就能学会
2. **纯文本** - 任何编辑器都能打开
3. **可版本控制** - 配合 Git 使用很方便
4. **专注内容** - 不用操心样式和排版
